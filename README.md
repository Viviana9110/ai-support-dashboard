# AI Support Dashboard

A modern AI-powered customer support dashboard built with Next.js, React, TypeScript and Tailwind CSS.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query
- Zustand
- React Hook Form
- Zod
- Axios
- Recharts

## Features (Planned)

- Authentication
- Dashboard
- AI Chat
- Tickets
- Customers
- Analytics
- Responsive Design
- Dark Mode

## Getting Started

```bash
npm install
npm run dev
```
